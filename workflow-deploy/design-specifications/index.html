Enhanced Design Specifications for FocusLoop

Design Philosophy

The enhanced FocusLoop design will embody Apple's design principles while creating a distinctive, premium experience specifically tailored for executives with ADHD. The design will focus on:

  1. Clarity: Distraction-free interfaces with purposeful visual elements
  2. Deference: Content-first approach where the interface recedes to highlight tasks
  3. Depth: Thoughtful use of layers, animations, and transitions to create a sense of space

Color System

Primary Mode Colors

  • Planner Mode: Gradient from #34C759 (Apple green) to #30B350, with subtle variations for depth
  • Execution Mode: Gradient from #FF3B30 (Apple red) to #E53730, with intensity shifts for focus states
  • Review Mode: Gradient from #AF52DE (Apple purple) to #9F4ACA, with tints for data visualization

Supporting Colors

  • Neutral Palette: System grays (6 levels) for text, backgrounds, and UI elements
  • Accent Colors: System blue (#007AFF) for interactive elements
  • Success/Error States: System green/red for feedback
  • Dark Mode Variants: Automatically adjusted color values for dark mode

Typography

Font System

  • Primary Font: SF Pro Display and SF Pro Text
  • Headings: SF Pro Display Medium/Bold with -0.5px tracking
  • Body Text: SF Pro Text Regular with 0px tracking
  • UI Elements: SF Pro Text Medium with appropriate optical sizing
  • Dynamic Type Support: All text scales appropriately with system text size settings

Text Hierarchy

  • Primary Headings: 24pt SF Pro Display Bold
  • Secondary Headings: 20pt SF Pro Display Medium
  • Tertiary Headings: 17pt SF Pro Display Medium
  • Body Text: 17pt SF Pro Text Regular
  • Secondary Text: 15pt SF Pro Text Regular
  • Caption Text: 13pt SF Pro Text Regular
  • Leading (Line Height): 1.3× font size for optimal readability

Layout System

Grid System

  • Base Unit: 8px grid for all spacing and layout decisions
  • Margins: 16px standard screen edge margin, 24px on larger displays
  • Padding: 16px standard internal padding
  • Card Spacing: 16px between cards, 24px between sections

Component Spacing

  • Vertical Rhythm: 8px, 16px, 24px, and 32px spacing options
  • Element Grouping: Related elements spaced at 8px, groups at 16px or 24px
  • Content Density: Optimized for information display while maintaining breathing room

Responsive Behavior

  • Breakpoints: Small (320-375px), Medium (376-744px), Large (745px+)
  • Layout Adaptation: Single column on small, two columns on medium, multi-column on large
  • Component Scaling: Key UI elements scale proportionally across breakpoints

Component Design

Cards

  • Corner Radius: 12px for all cards
  • Shadow: Subtle layered shadows (y-offset: 2px, blur: 8px, opacity: 8%)
  • Border: 0.5px stroke in system gray for light mode, darker gray for dark mode
  • Internal Padding: 16px standard, 20px for featured cards
  • States: Rest, Hover, Active, and Selected states with appropriate visual feedback

Buttons

  • Primary Buttons: Pill-shaped (fully rounded ends), filled with mode color
  • Secondary Buttons: Pill-shaped with stroke and mode color text
  • Tertiary Buttons: Text-only with mode color
  • Icon Buttons: Circular with 40px diameter for touch targets
  • States: Rest, Hover, Active, and Disabled with appropriate visual feedback
  • Padding: 16px horizontal, 12px vertical for comfortable touch targets

Input Fields

  • Text Fields: Rounded rectangle (8px radius) with subtle background
  • Focus State: Highlighted border in mode color when active
  • Error State: System red border with error message
  • Placeholder Text: System gray, 70% opacity
  • Input Text: System black/white with 100% opacity

Navigation

  • Tab Bar: Custom designed with mode-specific indicators
  • Selection Indicator: Animated pill that slides between tabs
  • Icons: SF Symbols with weight matching adjacent text
  • Active State: Mode color for active tab, system gray for inactive

Animation and Motion

Micro-interactions

  • Button Feedback: Subtle scale (95%) and color shift on press
  • Input Focus: Smooth border transition when field is activated
  • Toggle States: Elegant transitions between on/off states
  • Loading States: Custom spinner using mode colors

Transitions

  • Mode Switching: Smooth cross-fade with coordinated color transitions (0.3s duration)
  • Card Expansion: Natural spring physics when expanding task details
  • List Animations: Staggered fade-in for task lists (0.05s delay between items)
  • Progress Indicators: Continuous motion for timers and progress bars

Custom Animations

  • Task Completion: Satisfying checkmark animation with particle effects
  • Timer Visualization: Breathing animation that subtly pulses during focus sessions
  • Achievement Moments: Celebratory animations for completed tasks and milestones

Enhanced UI Components

AI Interview Interface

  • Conversational UI: Chat-like interface with clear question/answer pairing
  • Progressive Disclosure: Questions appear one at a time with smooth transitions
  • Context Awareness: Visual indicators showing how current question relates to previous answers
  • Input Methods: Support for text, voice, and quick-select options
  • Thinking States: Elegant loading animations during AI processing

Task Breakdown Visualization

  • Hierarchical Display: Parent-child relationships clearly visualized
  • Time Allocation: Visual representation of estimated time for each subtask
  • Dependencies: Subtle connecting lines showing task relationships
  • Confidence Indicators: Visual representation of AI confidence in time estimates
  • Drag-and-Drop Reordering: Fluid interaction for rearranging tasks

Focus Timer

  • Immersive Mode: Full-screen timer with minimal distractions
  • Progress Visualization: Circular progress indicator with subtle animation
  • Time Remaining: Large, clear typography showing remaining time
  • Break Indicators: Visual cues for upcoming breaks
  • Distraction Blocking: Elegant UI for enabling focus mode

Performance Analytics

  • Data Visualization: Custom charts and graphs using Apple's design language
  • Comparative Views: Before/after visualizations of performance improvements
  • Trend Indicators: Subtle arrows and colors showing positive/negative trends
  • Insight Cards: Beautifully designed cards highlighting key insights
  • Interactive Elements: Touch-responsive charts that reveal additional details

Mode-Specific Enhancements

Planner Mode (Green)

  • Task Input Flow: Redesigned with conversational AI interface
  • Smart Suggestions: Elegantly presented AI task recommendations
  • Time Blocking: Visual calendar interface for scheduling tasks
  • Priority Visualization: Clear visual hierarchy for task importance
  • Context Tagging: Simple system for categorizing tasks by context

Execution Mode (Red)

  • Distraction-Free Focus: Immersive interface that eliminates unnecessary elements
  • Real-Time Progress: Subtle indicators showing progress against estimates
  • Mind Dump Feature: Elegant interface for capturing stray thoughts
  • Focus Aids: Optional background sounds and visual cues for maintaining focus
  • Break Timer: Well-designed break reminder system

Review Mode (Purple)

  • Performance Dashboard: Beautifully visualized performance metrics
  • Insight Generation: AI-generated insights presented in elegant cards
  • Trend Analysis: Sophisticated but clear visualization of performance trends
  • Achievement Tracking: Visual celebration of accomplishments
  • Improvement Suggestions: Tactfully presented recommendations

Accessibility Enhancements

  • Color Contrast: All text meets WCAG AA standards (4.5:1 for normal text, 3:1 for large text)
  • Dynamic Type: All text scales appropriately with system settings
  • VoiceOver Support: Properly labeled elements for screen reader compatibility
  • Reduced Motion: Alternative animations for users with motion sensitivity
  • Keyboard Navigation: Complete keyboard control for desktop use

Design System Implementation

  • Component Library: Comprehensive library of reusable components
  • Design Tokens: Centralized color, typography, and spacing variables
  • Dark Mode Support: Complete dark mode implementation
  • Responsive Rules: Clear guidelines for adaptation across screen sizes
  • Animation Library: Reusable animation patterns and timing functions

This enhanced design specification addresses the limitations of the current prototype while incorporating advanced AI capabilities into a cohesive, premium user experience that follows Apple's design principles while creating a unique identity for FocusLoop.
